# Under Construction Page

## Overview

A simple, branded "Under Construction" page for the Hooked on Phonics Game Publishing CMS.

## Features

- ✅ Official Hooked on Phonics logo
- ✅ Brand colors (light blue gradient background)
- ✅ Quicksand font (matching their site)
- ✅ Animated bouncing dots in brand colors
- ✅ Clean, friendly design
- ✅ Mobile responsive

## Design Elements

**Background:**
- Gradient from light blue (#C4E9EF) to white
- Matches Hooked on Phonics brand aesthetic

**Logo:**
- Official SVG logo from their website
- Responsive sizing (24/32 height on mobile/desktop)

**Badge:**
- "Under Construction" in yellow (#FFD700)
- Rounded pill shape
- Bold text

**Animation:**
- Four bouncing dots
- Brand colors: Blue, Orange, Yellow, Purple
- Staggered animation timing

**Typography:**
- Quicksand font (their brand font)
- Clear hierarchy
- Easy to read

## Viewing

```bash
npm run dev
# Visit http://localhost:5173
```

## Next Steps

When ready to build the actual CMS, simply replace `src/App.jsx` with the real application components. The design system, fonts, and colors are already configured.
